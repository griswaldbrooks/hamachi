# hamachi docs

Project documentation for `hamachi` — a hard fork of [nothinglabs/openmelt2](https://github.com/nothinglabs/openmelt2),
the Arduino-based melty-brain combat robot controller. Named "hamachi" because tuna melt.

## Index

- [Roadmap](./roadmap.md) — long-term plan for the fork
- [Competition 2026-06-06](./competition-2026-06-06.md) — short-term build plan
  - [BOM](./competition-2026-06-06-bom.md) — reproducible parts list with vendor part numbers
  - [Build Log](./competition-2026-06-06-build-log.md) — running log of prints, assembly, tuning, results
- [Architectural Decision Records](./decisions/) — durable decisions and their rationale

## Conventions

- Decisions go in `docs/decisions/NNNN-slug.md` using the MADR format.
- This tree is intentionally docusaurus-compatible — when the docs site lands, this becomes the `docs/` source root.
- Issue tracking: GitHub Issues for human-visible roadmap items; [beads](https://github.com/steveyegge/beads) (`bd`) for agent task/memory state.
